# QUICK START GUIDE - For Complete Beginners

If you've never built a website before, follow these exact steps:

## Step 1: Get Web Hosting (Choose ONE option)

### Option A: Free Hosting (Best to Start)
**GitHub Pages (Recommended for beginners)**
1. Go to https://github.com and create a free account
2. Create a new repository called "yourname-glp1-site"
3. Upload all your files (index.html, styles.css, script.js)
4. Go to Settings ‚Üí Pages ‚Üí Enable GitHub Pages
5. Your site will be live at: yourname.github.io/yourname-glp1-site

**Netlify (Also Free & Easy)**
1. Go to https://www.netlify.com
2. Sign up for free account
3. Drag and drop your files
4. Site goes live instantly
5. Get a custom domain later

### Option B: Paid Hosting (~$3-10/month)
**Recommended hosts:**
- Namecheap (cheapest): $2.88/month
- Bluehost: $2.95/month (first year)
- HostGator: $2.75/month (first year)
- SiteGround: $3.99/month

**What to do:**
1. Sign up and buy hosting plan
2. Get a domain name (yoursite.com) - usually included first year
3. Use their "File Manager" or FTP to upload your files
4. Put files in "public_html" folder

## Step 2: Replace Affiliate Links

1. Open `index.html` in a text editor (Notepad on Windows, TextEdit on Mac)
2. Press Ctrl+F (Cmd+F on Mac) to open Find
3. Search for: `YOUR_AFFILIATE_LINK_1`
4. Replace with your actual affiliate link
5. Repeat for `YOUR_AFFILIATE_LINK_2` and `YOUR_AFFILIATE_LINK_3`
6. Save the file

**Don't have affiliate links yet?** 
- Keep the placeholder links for now
- Apply to programs (see Step 4)
- Update links once approved (takes 1-7 days)

## Step 3: Customize Your Site

### Change Site Name:
Find this line in `index.html`:
```html
<div class="logo">GLP-1 Compare</div>
```
Change to your site name:
```html
<div class="logo">Your Site Name</div>
```

### Update Provider Names:
Search for "Provider 1" and replace with actual company name (like "Hims")
Do the same for Provider 2 and Provider 3

### Update Prices:
Find `$297/month` and change to real prices from the actual programs

## Step 6: Track Your Success

### Install Google Analytics
1. Go to https://analytics.google.com
2. Create free account
3. Get tracking code
4. Add to `index.html` before `</head>` tag:

```html
<!-- Google Analytics -->
<script async src="https://www.googletagmanager.com/gtag/js?id=YOUR-ID"></script>
<script>
  window.dataLayer = window.dataLayer || [];
  function gtag(){dataLayer.push(arguments);}
  gtag('js', new Date());
  gtag('config', 'YOUR-ID');
</script>
```

Replace `YOUR-ID` with your actual tracking ID

### What to Track:
- Daily visitors
- Which pages get most traffic
- Where visitors come from
- How many click your affiliate links
